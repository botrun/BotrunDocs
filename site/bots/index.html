波特小隊

波特小隊是 Botrun 平台的預建專業 AI 角色群。每個角色針對特定領域最佳化，使用者可依需求選擇。

除了預建角色，使用者可透過 Hatcher（Bot 建立器）建立完全自訂的 Bot。

角色一覽

角色 專長領域 適用情境
波庫 SQL 資料庫查詢 撰寫 SQL 語法、資料庫查詢、資料分析
波程 程式碼撰寫 各類程式語言開發、除錯
波秒答 快速問答 即時回答、一般諮詢
波文 PDF 自然語言檢索 文件查詢、PDF 擷取、摘要

波媽 提示工程與自訂 Bot 建立自訂波特人、提示詞設計

在平台上可依分類篩選：寫 SQL 資料庫 / 寫程式 / 立即回覆 / PDF 檢索 / 數據分析 / 提示工程。

各角色詳細說明

波庫 — SQL 資料庫查詢

專門處理 SQL 相關任務。以自然語言描述需求，波庫自動產生對應 SQL 語法。

案例： 內政部專案，將 SQL 開發時間從兩天縮短至三小時，產出少子化根因分析、住宅與婚齡關聯性等分析。

波程 — 程式碼撰寫

協助各類程式語言開發，包含程式碼撰寫、除錯、邏輯解說。

波秒答 — 快速問答

針對一般性問題提供即時回答，適合不需要特定知識庫的快速諮詢。

波文 — PDF 自然語言檢索

處理文件類任務。上傳 PDF 後用自然語言提問，波文從文件中找到相關段落並回答，附帶引證來源。

案例： 政策研究輔助，3000 份專業文件知識庫，回答品質達專業水準。

波分 — 數據分析與視覺化

處理數據分析任務：資料清理、統計分析、圖表繪製。使非程式背景人員也能完成專業數據分析。

案例： 波地圖專案，協助地理資訊數據分析與地圖繪製。

波媽 — 提示工程與自訂 Bot

協助設計提示詞（Prompt），並引導建立自訂 Bot。儲存的提示詞模板可重複使用。

自訂 Bot（Hatcher）

使用者可透過 Hatcher 建立完全自訂的 Bot，設定項目包含：

項目 說明
Bot 名稱 自訂名稱與描述
基礎模型 從多個提供商選擇 AI 模型
代理模型 多代理模式下的模型選擇
提示詞模板 行為指令與角色定義
知識庫 上傳 PDF/DOCX/CSV 建立專屬知識
MCP 工具 選擇外部工具組合
運作模式 基礎模式或多代理模式

管理功能

  • 建立 / 編輯 / 刪除 Bot
  • 上傳 / 刪除知識庫檔案
  • 分享 Bot 給其他使用者
  • 取消分享
  • 設為預設 Bot

Hatcher 路徑：/hatcher（Bot 列表）、/hatcher/editor/:hatchId（編輯）
