波特人功能總覽

波特人（Botrun）的功能分為三個層次，點擊進入各層詳細說明：

第一層：原生波特人 — 進入就能用

不需任何設定，進入 botrun.ai 就有的能力：搜尋、PDF 分析、圖片辨識、AI 繪圖、圖表、網頁製作、語音輸入。外部工具呼叫約 3 秒，比 ChatGPT GPTs 快 5 倍。

詳細說明 → 原生波特人功能

第二層：孵化器（Hatcher）— 自己建 Bot

不用寫程式，填表單就能打造專屬 Bot：定義角色、上傳知識庫（RAG）、勾選 MCP 工具、開啟多代理模式。內政部「波孵」用它把簽證審查從 3 小時縮短至 30 秒。

詳細說明 → 孵化器功能

第三層：平台能力 — 分享、匯出、部署

Bot 分享給同事、API 串接到你的系統、.botrun 編譯匯出成 Python、地端部署資料不出內網。

詳細說明 → 平台能力
