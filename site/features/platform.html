分享與協作

Bot 分享

功能 說明
獨佔 URL 每個 Bot 有唯一連結（/b/BotID），複製就能分享
指定分享 分享給特定使用者
取消分享 隨時收回分享權限
My Playbook 瀏覽和複製他人的 Bot 設計
權限驗證 分享的 Bot 需通過權限檢查才能使用

對話分享

  • 點擊對話頂部的分享按鈕
  • 複製分享連結（會顯示「已複製分享連結！」）
  • 對方打開連結即可看到完整對話（唯讀，路由 /s/:sessionId）

.botrun 編譯匯出

在波特人設計好的 Bot，匯出成 Python，放到你自己的伺服器跑。

  • 編譯為 Python：.botrun 檔案可編譯為 .py 腳本
  • OpenAI API 相容：編譯後程式碼可使用 OpenAI API 格式呼叫
  • litellm 相容：支援任意模型提供商（不限於 OpenAI）
  • 無平台鎖定：編譯後可脫離 Botrun 平台獨立運行
  • 版本控制：提示詞像程式碼一樣管理，支援 Git 追蹤

地端部署

裝在你的機房裡，資料完全不出內網。

方式 說明
雲端平台 botrun.ai，網頁直接使用
地端部署 客戶自有機房，Docker 容器化部署

台灣三中心部署

波特人在台灣設有三個資料中心，確保資料主權與服務穩定：

地點 基礎設施 模型
臺北 AWS Anthropic Claude Sonnet 4
新竹 國網中心 Llama-4-Maverick, OpenAI gpt-oss 120B
彰化 Google Anthropic Claude Haiku 4.5

部署特色

  • Docker-in-Docker 隔離：每個使用者獨立的容器環境
  • 無外網依賴：所有資料處理在內網完成
  • 符合政府資安規範：通過多個部會資安審查
  • 資料完全不出內網：符合「行政院 AI 指引」的封閉雲端要求

帳號與設定

登入方式

方式 說明
Email + 密碼 Firebase 帳號登入
Google OAuth Google 帳號一鍵登入
SSO 整合身份系統
MFA 多因子驗證 TOTP 驗證，可強制啟用
外部存取 Token 驗證（LINE 等外部平台）

個人設定

設定 說明
語音模型 選擇轉文字的 AI 模型（4 種可選）
介面語言 繁體中文（zh-Hant）/ English
閃電語音 錄音完自動發送
返回警告 離開對話前確認
深淺色主題 深色或淺色介面（Cookie 儲存）

帳號資訊（唯讀）

  • 會員帳號、所屬機關/部門
  • 會員啟用日、到期日
  • 前端/後端版本號

Token 用量追蹤

  • 全域 Token 使用統計
  • 用量百分比顯示
  • 最後更新時間標記
  • 詳細用量分析連結

  • 近期對話列表：重新命名、刪除、選項管理
  • 新增對話按鈕

系統公告
