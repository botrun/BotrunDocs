# 波特人功能總覽

波特人（Botrun）的功能分為三個層次，點擊進入各層詳細說明：

---

## 第一層：原生波特人 — 進入就能用

不需任何設定，進入 botrun.ai 就有的能力：搜尋、PDF 分析、圖片辨識、AI 繪圖、圖表、網頁製作、語音輸入。外部工具呼叫約 3 秒，比 ChatGPT GPTs 快 5 倍。

> 詳細說明 → [原生波特人功能](native.html)

---

## 第二層：孵化器（Hatcher）— 自己建 Bot

不用寫程式，填表單就能打造專屬 Bot：定義角色、上傳知識庫（RAG）、勾選 MCP 工具、開啟多代理模式。內政部「波孵」用它把簽證審查從 3 小時縮短至 30 秒。

> 詳細說明 → [孵化器功能](hatcher.html)

---

## 第三層：平台能力 — 分享、匯出、部署

Bot 分享給同事、API 串接到你的系統、.botrun 編譯匯出成 Python、地端部署資料不出內網。

> 詳細說明 → [平台能力](platform.html)

---

## 19 個波特人 Bot

波特人平台提供 19 個專業 Bot，涵蓋六大分類：

### 原生 Bot

| 名稱 | 功能 |
|------|------|
| 原生波特人 | 任意對話、呼叫其他 Bot、多代理工具 |

### 聲音與語言精靈

| 名稱 | 功能 |
|------|------|
| 波逐字稿 | 音檔轉文字、紀錄摘要 |
| 波翻譯 | 多國語言翻譯 |
| 波臺德 | 直接與臺德語言模型對談 |
| 波Youtube | Youtube 字幕轉逐字稿與摘要 |

### 文字魔法師

| 名稱 | 功能 |
|------|------|
| 波分段 | 長段落分段整理重點 |
| 波新聞稿 | 模仿範例撰寫新聞稿 |
| 波公文 | 公文撰寫、除錯、潤飾 |
| 波提示工程 | 優化提示工程內容 |
| 波簡報 | 產生精彩簡報 |

### 創意畫匠

| 名稱 | 功能 |
|------|------|
| 波繪圖 | 文字生成圖片 |
| 波畫室 | 引導式精緻繪圖 |
| 波流程圖 | 心智圖、流程圖、甘特圖 |
| 波網頁 | 設計網頁並產生連結 |

### 文件與數據

| 名稱 | 功能 |
|------|------|
| 波文件問答 | 多格式文件上傳問答 |
| 波數據分析 | CSV/Excel 數據視覺化 |
| 波會議記錄 | 文字整理成會議紀錄 |
| 波旅程地圖 | 機關痛點旅程地圖報告 |

### 程式碼

| 名稱 | 功能 |
|------|------|
| 波程 | 程式碼瀏覽與問題解決 |
