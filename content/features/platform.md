## 分享與協作

### Bot 分享

| 功能 | 說明 |
|------|------|
| 獨佔 URL | 每個 Bot 有唯一連結（`/b/BotID`），複製就能分享 |
| 指定分享 | 分享給特定使用者 |
| 取消分享 | 隨時收回分享權限 |
| My Playbook | 瀏覽和複製他人的 Bot 設計 |
| 權限驗證 | 分享的 Bot 需通過權限檢查才能使用 |

### 對話分享

- 點擊對話頂部的分享按鈕
- 複製分享連結（會顯示「已複製分享連結！」）
- 對方打開連結即可看到完整對話（唯讀，路由 `/s/:sessionId`）

## API 串接

> **把波特人的能力嵌入你自己的系統。**

### 可用的 API 端點

| 類別 | API | 方法 | 功能 |
|------|-----|------|------|
| **對話** | `/v2/chat/{sessionId}` | GET (SSE) | 與 Bot 聊天（串流） |
| | `/v2/chat/{sessionId}/stop` | POST | 停止回應 |
| | `/v2/chat/{sessionId}/stream` | GET (SSE) | 恢復中斷的串流 |
| **Bot 管理** | `/v2/hatch/create` | POST | 建立新 Bot |
| | `/v2/hatch/{hatchId}` | GET | 取得 Bot 設定 |
| | `/v2/hatch` | PUT | 更新 Bot |
| | `/v2/hatch` | DELETE | 刪除 Bot |
| | `/v2/hatch/list` | GET | 列出所有自建 Bot |
| **知識庫** | `/v2/hatch/upload` | POST | 上傳知識庫檔案 |
| | `/v2/hatch/file` | DELETE | 刪除知識庫檔案 |
| **模型與工具** | `/v2/hatch/models` | GET | 取得可用模型列表 |
| | `/v2/hatch/mcp-tools` | GET | 取得可用 MCP 工具 |
| **檔案** | `/v2/upload` | POST | 上傳對話檔案 |
| | `/v2/speech-to-text` | POST | 語音轉文字 |
| **使用者** | `/user/info` | GET | 取得帳號資訊 |
| | `/user/bots/all` | GET | 取得所有 Bot（含釘選、自建、共享） |
| **用量** | `/v2/info/token/usage` | GET | 查詢 Token 使用量 |

所有 API 需要 Bearer Token 認證。

## .botrun 編譯匯出

> **在波特人設計好的 Bot，匯出成 Python，放到你自己的伺服器跑。**

- **編譯為 Python**：`.botrun` 檔案可編譯為 `.py` 腳本
- **OpenAI API 相容**：編譯後程式碼可使用 OpenAI API 格式呼叫
- **litellm 相容**：支援任意模型提供商（不限於 OpenAI）
- **無平台鎖定**：編譯後可脫離 Botrun 平台獨立運行
- **版本控制**：提示詞像程式碼一樣管理，支援 Git 追蹤

## 地端部署

> **裝在你的機房裡，資料完全不出內網。**

| 方式 | 說明 |
|------|------|
| **雲端平台** | botrun.ai，網頁直接使用 |
| **地端部署** | 客戶自有機房，Docker 容器化部署 |

### 部署特色

- **Docker-in-Docker 隔離**：每個使用者獨立的容器環境
- **無外網依賴**：所有資料處理在內網完成
- **符合政府資安規範**：通過多個部會資安審查
- **資料完全不出內網**：符合「行政院 AI 指引」的封閉雲端要求

## 帳號與設定

### 登入方式

| 方式 | 說明 |
|------|------|
| Email + 密碼 | Firebase 帳號登入 |
| Google OAuth | Google 帳號一鍵登入 |
| 企業 SSO | 整合企業身份系統 |
| MFA 多因子驗證 | TOTP 驗證，可強制啟用 |
| 外部存取 | Token 驗證（LINE 等外部平台） |

### 個人設定

| 設定 | 說明 |
|------|------|
| 語音模型 | 選擇轉文字的 AI 模型（4 種可選） |
| 介面語言 | 繁體中文（zh-Hant）/ English |
| 閃電語音 | 錄音完自動發送 |
| 返回警告 | 離開對話前確認 |
| 深淺色主題 | 深色或淺色介面（Cookie 儲存） |

### 帳號資訊（唯讀）

- 會員帳號、所屬機關/部門
- 會員啟用日、到期日
- 前端/後端版本號

### Token 用量追蹤

- 全域 Token 使用統計
- 用量百分比顯示
- 最後更新時間標記
- 詳細用量分析連結

## 介面功能

### 側邊欄

- Bot 池：顯示前 6 個 Bot（支援釘選、顯示更多）
- 近期對話列表：重新命名、刪除、選項管理
- 新增對話按鈕

### 系統公告

平台可發布公開公告，使用者登入後看到最新資訊。

### 系統訊息

| 訊息 | 觸發條件 |
|------|---------|
| 檔案過大 | 上傳超過 200MB |
| 上傳成功/失敗 | 檔案上傳結果 |
| 語音無聲 | 錄音沒有聲音 |
| 語音轉換失敗 | 轉文字失敗（可下載音檔重試） |
| 字數超限 | 超過每小時字數限制 |
